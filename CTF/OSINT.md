# [recon-ng](https://github.com/lanmaster53/recon-ng)

## entering recon-ng
```bash
recon-ng
```

## Searching for Modules
```bash
modules search <module name>
```

## Installing Modules
```bash
marketplace install <module name>
```

## Using a Module
```bash
module load <module name>
```

### Set Options
```bash
options set <option> <value>
```

### Run module
```bash
run
```
